Classy-JSON

Classy-JSON is a Python library that aims to allow for easier usage of Python dictionaries by adding dot-accessible dictionaries. By subclassing a normal Python dictionary and replacing the functions relating to attribute setting and getting, Classy-JSON is able implement this in an efficient manner.
